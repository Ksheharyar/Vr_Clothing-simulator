👕🕶️ VR Clothing Simulator

A Virtual Reality Clothing Try-On Simulation developed in Unreal Engine 5 for Meta Quest devices.
This project allows users to interact with a virtual showroom environment and dynamically change outfits in real time using immersive VR interactions. 🚀

🌟 Project Overview

The VR Clothing Simulator is designed to provide an interactive virtual fashion experience where users can browse, select, and wear different clothing items inside a fully immersive VR environment. 🛍️✨

The simulation focuses on:

👔 Real-time outfit switching
🎮 Interactive VR controls
🏬 Immersive showroom experience
🪞 Mirror-based outfit visualization
🖥️ VR UI interactions
⚡ Optimized VR performance for Meta Quest
🚀 Features
👕 Real-Time Clothing Switching

Users can dynamically change:

👔 Shirts
👖 Pants
👟 Shoes
🧢 Hats

using interactive VR buttons in real time.

🎮 VR Interaction System
🕹️ Motion controller interaction
🔘 Physical VR button system
📡 Ray-based interaction mechanics
✨ Smooth and immersive VR experience
🪞 Mirror Visualization

Players can view their character in a virtual mirror after changing outfits, improving realism and immersion. 🌟

🏬 Custom Showroom Environment

The project uses a showroom-style environment designed to simulate a virtual fashion store experience. 🛍️✨

🧑‍🚀 MetaHuman Integration

The simulation uses a MetaHuman-based character setup for realistic character appearance and high-quality clothing presentation. 🎭

⚡ Optimized for VR

Performance optimization techniques include:

📉 Reduced polygon usage
🎨 Optimized materials
💡 Efficient lighting setup
🚀 VR-friendly rendering settings
🎯 Stable frame rate for smooth gameplay
